Web Development Concepts (jQuery Accordion)

Frontend Development

This is the art of creating what users see and interact with directly in their web browser. It involves HTML for structure, CSS for styling, and JavaScript for interactivity, ensuring a seamless and engaging user experience. Key frameworks like React, Angular, and Vue.js are often used.

Backend Development

The powerhouse behind the scenes, backend development manages server-side logic, databases, APIs, and ensures the website functions correctly. Technologies include Node.js, Python (Django/Flask), Ruby on Rails, Java (Spring), and PHP (Laravel), along with databases like PostgreSQL, MySQL, and MongoDB.

Full-Stack Development

A full-stack developer is a versatile expert proficient in both frontend and backend technologies. They can build and manage entire web applications from the user interface to the server-side infrastructure, understanding the complete development lifecycle.

Web APIs (Application Programming Interfaces)

APIs are crucial for modern web development, allowing different software applications to communicate and exchange data. They enable integration of third-party services (like maps, payment gateways, or social media) and power dynamic, feature-rich websites. This weather demo itself uses an API!

Behind the Code: About This Page

Welcome! This single-page website began as a project for my GIT418 course at ASU and has since evolved into a personal showcase of modern web development techniques. I've particularly enjoyed enhancing its interactivity and ensuring a responsive experience across devices. Here's a peek at what this page demonstrates:

  • cloud Live Weather Reports: Fetching and displaying real-time data from the OpenWeatherMap API.
  • palette Theme Customization: Utilizing Web Storage to remember your theme preferences between visits.
  • widgets Interactive jQuery UI Widget: Using the accordion widget for neatly organized content.
  • photo_library JavaScript Slideshow: A custom-built carousel to showcase visual content.
